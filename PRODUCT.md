# SalesFlow CRM

## Product Purpose
CRM + Planner สำหรับตัวแทนประกันชีวิตและนักขายเดี่ยวในไทย เชื่อม Contact + Task + Calendar + Goal เพื่อวัดว่าเดือนนี้ทำนัดได้ตามเป้าหมายไหม

## Register
product

## Users
- ตัวแทนประกันชีวิต (AIA, Muang Thai Life) — ใช้บน desktop ตอนเช้าก่อนออกไปหาลูกค้า และบน mobile ขณะเดินทาง
- นักขายเดี่ยว solo seller — financial advisor, real estate agent, coach

**Physical scene**: ตัวแทนประกันนั่งที่โต๊ะทำงานตอน 8 โมงเช้า แสงธรรมชาติ เปิด laptop ดู dashboard ว่าวันนี้ต้องโทรหาใคร ก่อนออกไปพบลูกค้า

## Brand
- สี: น้ำเงิน (blue) + ขาว — ดูน่าเชื่อถือ professional trustworthy
- ไม่ใช้ emoji เป็น icons — SVG line icons เท่านั้น
- ภาษาไทยเป็นหลัก
- Tone: confident, professional, calm — ไม่เล่น ไม่สนุกสนาน

## Anti-references
- Salesforce: ซับซ้อนเกินไป ดูหนัก
- HubSpot: มีฟีเจอร์เยอะ ดู enterprise เกิน
- Notion: too minimal ไม่มี structure ชัดพอสำหรับ sales workflow
- LINE OA dashboards: สีสันเกินไป ดูไม่ professional

## Strategic Principles
- เบา เร็ว ใช้ได้ทันที ไม่ต้อง setup เยอะ
- ทุก action ต้องทำได้ใน 2-3 คลิก
- Goal progress ต้องเห็นทันทีบน dashboard
- ไม่แข่งความครบ แข่งความเบา + goal integration
